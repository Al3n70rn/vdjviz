About

VDJviz is a lightweight immune repertoire browser. After registering and signing in, users can upload clonotype tables that were produced using MiTCR, MiGEC and IgBlastWrapper Rep-Seq processing software. VDJviz server will compute a set of key repertoire statistics and display them as interactive graphics.

This application was built using Play Framework and VDJtools API. VDJviz can also be run as a local server, check out the link to source code below.

Some useful links:

https://github.com/mikessh/vdjviz - VDJviz source code

https://github.com/mikessh/vdjtools - VDJtools source code

https://github.com/mikessh/vdjtools/wiki/Input - detailed description of input formats

https://github.com/mikessh/vdjtools/tree/master/examples/aging_lite/samples sample datasets

https://github.com/mikessh/vdjtools/wiki/Modules - detailed description of post-analysis
